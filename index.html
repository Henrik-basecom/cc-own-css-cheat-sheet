HTML Tags

A Table with explainations about html tables tags.
Tag Name Description
<table> Table wraps the other table elements
<thead> Table Head wraps the header rows that have infos about the columns
<tbody> Table body wraps the main data of the table
<tfoot> Table Foot wraps the conlusion, extra infos or a summary of the table
<caption> Table Caption gives the table a caption, for an accessible description
<tr> Table Row defines a row of cells in a table
<th> Table Header defines a cell as a header of the other cells
<td> Table Data Cell defines a cell inside the table
